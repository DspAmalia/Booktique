Aplicatia este usor de folosit. Folderul git contine aplicatia propriu-zisa si un folder numit "Database" unde poate fi gasit un script de sql (ce contine atat tabelele, cat si datele din tabele). Acesta trebuie rulat in SQL Server Management Studio pentru a putea avea acces la baza de date.

Aplicatia Booktique, dupa ce este pusa pe PC, este gata de utilizare. Userul trebuie sa isi creeze un cont pentru a avea mai multe facilitati (acces la favorite, cos de cumparaturi) si, de asemenea, achizitionarea unui abonament deblocheaza meniuri noi.

Achizitionarea unui abonament nu implica cumparare cu bani reali, ci este construit pe baza unei chei de test. Astfel, userul trebuie sa introduca date fictive in textboxurile aferente datelor cardului, iar la numarul cardului acesta trebuie sa introduca "4242 4242 4242 4242".
